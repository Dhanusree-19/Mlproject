This project predicts whether a customer is likely to **churn (leave a service)** using Machine Learning.  
It is built as a **complete end-to-end ML project** with a web interface.

---

## What is Churn Prediction?

Customer churn prediction helps businesses identify customers who are likely to stop using their service.  
By predicting churn early, companies can take actions to **retain customers**.

---

## About This Project

In this project, I built a full machine learning pipeline:
- Data ingestion
- Data preprocessing
- Model training & evaluation
- Prediction pipeline
- Web application using Flask

The model outputs:
- **Churn / No Churn**
- **Churn probability (%)**
